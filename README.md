# agent-dx.com

Your CLI was built for humans. Agents use it differently. They don't read docs,
don't have muscle memory, and hallucinate flags that don't exist.

This site is about the patterns that help: structured I/O, runtime schemas, input
hardening, skill files, and a few more. Eight principles total, each with code
examples.

https://agent-dx.com

## Stack

Next.js 15 + Nextra 4, static export to GitHub Pages.

## Development

```
npm install
npm run dev
```
